## Run Locally

Clone the project

```bash
  git clone https://github.com/main1479/monsters-rolodex.git $PROJECT_NAME
```

Go to the project directory

```bash
  cd $PROJECT_NAME
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
